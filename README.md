## 🚀 セットアップ手順・Secrets設定

- 必要なAPIキーは`GITHUB_TOKEN`（GitHub Actions用トークン）のみです。
- `.github/workflows/`配下のYAMLファイルやREADME.mdは、ユーザー名や個別IDの手動入力なしで動作します。
- 必要に応じてassetsディレクトリに画像を追加してください。
